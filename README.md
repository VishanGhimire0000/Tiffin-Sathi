# TiffinSathi – C2C Tiffin Marketplace

**TiffinSathi** is a **Customer-to-Customer platform** connecting **local cooks** with **students and office-goers** in Nepal. Cooks can list their tiffins, and customers can browse, order, and contact cooks directly.

---

## Features
- **Cooks:** Register, add/edit/delete tiffins, view orders  
- **Customers:** Browse/filter tiffins, place orders, rate/review  
- Simple, responsive design with images  

---

## Technology
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  

---

## Database Tables
- `users` – cooks/customers info  
- `tiffins` – meal listings  
- `orders` – order info  
- `reviews` – optional ratings/comments  
- `categories` – meal types  

---

## Installation
1. Clone the repo  
2. Import `tiffinsathi.sql` to MySQL  
3. Configure `config.php` with DB credentials  
4. Place project in local server (e.g., XAMPP `htdocs`)  
5. Access via `http://localhost/tiffinsathi`
